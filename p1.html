Many programming languages consist  of the machine instructions. There are programmable machines that use a series of unique instructions instead of programming languages generally.A programming language is a formal language which includes a set of instructions that produce different output types. The programming languages are used to implement algorithms in computer programming.The first was possibly the automatic flute player mentioned by the Musa brothers in Baghdad during the Islamic Golden Age in the 9th century[1] Since the early 1800s, programs have been used to control the actions of machines such as Jacquard looms, music boxes and player pianos.The programs for these devices (such as scrolls from a player piano) did not generate different actions in response to various inputs or conditions.Thousands of different programming languages were created, and more are created annually. Many programming languages are written in an imperative form ( i.e. as a sequence of operations to be performed) while other languages use the declarative form (i.e. you specify the desired result, not how to achieve it).
